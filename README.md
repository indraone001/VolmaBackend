# VolmaBackend
Backend for Volma App

## Project Setup
### Update dependency
`npm update`

## Knex Migration and Seeds
### make migration
`knex migrate:make migration_name`

### generate migration
`knex migrate:latest`

### generate rollback
`knex migrate:rollback`

### Seeding
`knex seed:make seed_name`

### Generate seed
`knex seed:run `
